PawPal AI

A polished pet-care scheduling assistant built from the Module 2 PawPal+ project. It retrieves care guidance, proposes tasks, checks conflicts, and explains its plan before changing the schedule.

View Demo Read Project README

What It Does

Owner confirmation required
Retrieves Care Guidance

Matches owner requests to local dog, cat, medication, grooming, senior-care, and schedule-safety guidance.

Plans Schedule Tasks

Turns retrieved guidance into concrete PawPal tasks with time, priority, frequency, and duration.

Checks Before Applying

Simulates the plan, flags exact-time conflicts, and only changes the real schedule after confirmation.
